CSS Box Model

Content-Box

Width/height includes only content. Padding and border are added outside.

Border-Box

Width/height includes content + padding + border, making sizing easier.

CSS Positioning

Static

Default. Follows normal flow.

Relative

Positioned relative to its normal position.

Absolute

Removed from normal flow, positioned relative to nearest ancestor.

Fixed

Stays fixed relative to viewport.

Sticky

Acts like relative until scroll reaches a threshold, then behaves like fixed.

Pseudo-elements

::before

Inserts content before element. Example: used for article markers.

::after

Inserts content after element.

::first-letter & ::first-line

Style first letter or first line of text.

Semantic HTML

Use meaningful tags to define content structure:


  • <header> - Top section
  • <nav> - Navigation links
  • <main> - Main content
  • <section> - Group of content
  • <article> - Independent content
  • <aside> - Related content / sidebar
  • <footer> - Footer
